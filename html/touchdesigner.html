TouchDesigner × MediaPipe: Vision-Based Interaction Primer & Workshop

This project explores alternative input modalities by building vision- and gesture-based interaction systems in TouchDesigner. Leveraging MediaPipe's real-time hand tracking, I route camera input to landmark data and translate that data into responsive controls—driving visuals, states, and effects with simple, intentional gestures.

By stepping beyond the cursor-and-touch paradigm, these prototypes aim to create interactions that are more direct, embodied, and immersive, demonstrating to me and others how computer vision can open up new UX patterns for creative tools and installations.

To make this knowledge transferable to others I designed expansive documentation that aspires to translate the esoteric knowledge of Node-based programming and Vision-based HCI to newcomers.

Progress

During my minor in Behaviour Design, we were given an individual six-week project to acquire a new skill or area of knowledge. Given the freedom to choose, I set out to learn TouchDesigner. I had been eyeing this program for a while, so there was no better time to start. That decision led to the following project: vision-based interaction in TouchDesigner using MediaPipe.

Through online resources and diligent study, I taught myself the basics of TouchDesigner and MediaPipe, and distilled this knowledge into a workshop, multiple demos, and a detailed document providing more in-depth guidance.
